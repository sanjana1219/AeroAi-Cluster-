# AeroAi Cluster

A Deep Dive into Airline Delays

Project Overview


Air transport plays a critical role in connecting people, businesses, and economies across the world. However, flight delays significantly impact airline operations, passenger satisfaction, and overall transportation efficiency.
This project analyzes real-world airline delay data using data mining techniques to identify delay patterns, major contributing factors, and performance differences among airlines. The study applies clustering methods to group airlines based on their delay characteristics and generate meaningful insights for improving operational efficiency.
